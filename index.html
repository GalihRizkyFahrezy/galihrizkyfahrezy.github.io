Source Code : github

Fitur : Create, Read, Update, Delete. upload image, limit image uploaded, pagination, search.

Halaman Utama

• Slider akan berubah selama 3 detik dan menampilkan 3 foto terbaru dari setiap berita yang dimasukan.
• Berita akan ditampilkan mulai dari yang terbaru terlebih dahulu.
• Search bar akan mencari berita berdasarkan judul.



Halaman Berita

Halaman berita menampilkan isi dari berita tersebut berupa judul, penulis, gambar dan juga konten dari berita tersebut.


Halaman Dashboard

Pada Halaman Dashboard berisi pagination yang membatasi daftar berita yang tampil sebanyak 5 berita serta proses untuk Create, Read, Update dan Delete.


Halaman Create

• Foto hanya akan menerima file .png dan .jpeg
• Foto hanya akan menerima file yang tidak lebih dari 100kb
• Content adalah variable bertipe LongText sehingga cocok untuk menulis konten yang diinginkan.


Halaman Update

• Update berita akan secara otomatis memasukan placeholder dari berita yang dipilih kedalam form
• Pada update berita jika foto tidak diisi maka akan tetap menyimpan foto yang sebelumnya.
